# Gmail / Foursquare like Pull to refresh solution for Android Titanium


![Image](../master/pulltorefresh.gif)

##Before you start
* Get the ti-smoothprogressbar module from [here](https://github.com/artanisdesign/ti-smoothprogressbar)
